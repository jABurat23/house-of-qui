## Features

### 📜 Technical Chronicles
The Scribe automatically "writes" the story of every project:
- **Module Inventory**: Lists all source files and identified components.
- **Imperial Timestamps**: Records exactly when the chronicle was last updated relative to the Imperial Calendar.
- **Authoritative Record**: Provides a single source of truth for operators to understand project scope at a glance.

### 📐 Architectural Blueprints (Mermaid)
Using the relationships mapped by the Cartographer, the Scribe generates valid Mermaid-js graph definitions:
- **Flow Visuals**: Shows how modules import or depend on each other.
- **Living Diagrams**: The blueprints update automatically whenever a new chronicle is generated.

### 🏛️ The Chronicler's Dashboard
A new high-tier panel on the Imperial Dashboard:
- **Project Registry**: A sidebar to browse all chronicled projects.
- **Scroll Viewer**: A dedicated reading area for markdown reports and code diagrams.
- **On-Demand Chronicling**: Operators with `OVERSEER` clearance can trigger an instant re-scan and documentation update.
