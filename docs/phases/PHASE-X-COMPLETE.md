## Features

### 🏛️ The Central Core (House of Qui)
The **House of Qui** is now the immutable center of the cartographic universe. 
- **Fixed Position**: Always rendered at the center coordinates (400, 300) in the universe view.
- **Top-Tier Clearance**: Requires `OVERSEER` mandate to view its internal code architecture.

### 🌌 Dual-Mode Topology
- **Universe View**: Visualizes project-to-project communication and the "Imperial Bus" traffic.
- **Nexus (Code) View**: Upon authorization, drills down into a project's actual source files and their internal import relationships.

### 🔐 Imperial Role Hierarchy
Access to the Cartographer's data is strictly governed by the following clearance levels:
1. `VISITOR`: Basic universe visibility.
2. `OPERATOR`: Access to standard project details.
3. `ARCHIVIST`: Access to project code-maps.
4. `OVERSEER`: Full access, including the House of Qui Core.

### ⌨️ Real-Time Code Inspection
- **Interactive Nodes**: Click on any source file to open the **Imperial Code Viewer**.
- **Auto-Discovery**: The scanner automatically identifies file relationships (imports/exports) and visualizes them as edges.
- **Self-Healing Design**: The graph dynamically adjusts as files are added or moved in the workspace.
